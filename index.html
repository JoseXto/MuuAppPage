¿Qué caracteristica tiene?

● Registro de usuarios ● Inicio de sesión ● Registro del rancho, granja o finca ● Control del ganado ● Control de vacunas y tratamientos

¿Qué beneficio tiene?

Que podra anexar algun trabajador a la misma cuenta.

¿Porque se decidio hacer esta app?

Los ganaderos necesitan llevar el control de sus animales, para ello, la mayoría de las personas usan tarjetones en dónde escriben todos los datos de cada animal como su identificador, nombre, fecha de nacimiento, edad, peso, raza, propósito e indicar si ha sido cría de otro animal del rancho. Escribir todos los datos por cada animal se vuelve tedioso, además que si se requiere ver los datos de uno en específico o ver su historial se necesita buscar entre todos los tarjetones que existen para encontrar el o los tarjetones deseados. Estos documentos se pueden perder con facilidad ya sea por descuido de un trabajador o por algún caso ajeno como puede ser un desastre natural, también con el tiempo los datos escritos se vuelven ilegibles por el desgaste del papel.

Decidimos actuar

Ante la problemática antes descrita, hemos pensado en una manera de agilizar el proceso aplicando la tecnología, para ello ideamos una aplicación para smartphones que venga a sustituir los tarjetones que se usan para escribir los datos del ganado.
